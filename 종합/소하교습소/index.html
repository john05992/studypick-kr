소하 교습소
따라서 아이가 문제를 풀기 전에 ‘예상 질문’을 스스로 만들어보고 그에 미리 답을 연습하는 말투를 사용하게 하며, “이 단원에서 선생님이 어떤 문제를 낼 것 같아?” “너라면 어떻게 물어볼 거야?”라고 유도한다. 소하 교습소은 예를 들어 미분 연산에서 자주 틀리는 실수 유형인 ‘합성함수의 도함수를 적용하는 것을 깜빡함’을 직접 노트에 정리하고, 해당 유형의 문제를 반복 풀도록 하는 것이다. 이때 학습 속도는 아들의 현재 수준에 맞춰 조절하며, 지나치게 어렵거나 많은 양을 주기보다는, 숙달된 후 다음 단계로 진입하게 유도한다. 더 나아가 문제를 풀 때는 지문의 질문 범위를 넘어서 보기나 서술문 전체에서 배경 지식과 확장 정보를 찾아내는 훈련을 반복한다. 소하 교습소은 예제와 연습문제 사이의 연결이 자연스럽지 않다면 학습은 점차 단기 기억에 의존하게 되고, 심지어 유사 문제라도 조금만 형태가 달라져도 새로운 문제로 인식된다. 목표와 실천 사이의 간극을 최소화하도록 지원하기 위해, 수업 직후 즉시 복습을 실시해 장기 기억 전환율을 높인다. 예를 들어 과학 시간에 배운 ‘생태계’ 단원에 대해 다큐멘터리 영상, 신문 기사, 백과사전 조사를 추가로 찾아보며 배경 지식을 넓히면, 시험 문제의 맥락을 더 정확히 이해할 수 있습니다.
